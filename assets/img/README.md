## Cara unggah lewat GitHub
1. Buka folder ini di repo: **Add file → Upload files**
2. Seret kelima foto, lalu pastikan namanya sama persis dengan tabel di atas (huruf kecil semua, ekstensi `.jpg`).
3. Klik **Commit changes**. Dalam ±1 menit foto langsung tampil di website.

> Tips ringan & cepat: kompres foto ke lebar maksimal ~1600px dan ukuran < 300 KB per file
> (mis. via squoosh.app atau tinypng.com) agar website tetap ringan dibuka.

Jika ingin memakai nama file lain, beri tahu kami — path-nya diatur di `assets/js/data.js`
(objek `IMG`), `assets/js/gallery.js`, `assets/js/home.js`, dan `tentang.html`.
